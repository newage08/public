# FesTimeBuilder

2025版/2024版の実タイムスロットを表示し、アーティストを手動ドラッグ&ドロップで配置するアプリです。iPhone（Safari）での操作を前提に、指ドラッグやタップ編集にも対応しています。

## 使い方
1. フェスを選ぶ（SUMMER SONIC TOKYO / FUJI ROCK FESTIVAL）
2. 年（2025版/2024版）と日を選ぶ
3. その年の出演者プリセットが自動表示される
4. 必要なら編集して「札を作る」
5. 札を空スロットへドラッグ&ドロップ（PCはマウス、iPhoneは指ドラッグ）
6. ステージ見出しをドラッグして横並び順を入れ替える
7. 枠の時間は次のどちらかで変更する
   - 枠の時刻表示をタップして `HH:MM` 入力
   - 枠そのものを上下にドラッグして時刻移動
8. 「元ポスターを開く」から公式ページを別タブで開く
9. 「画像出力」でPNG保存（SNS投稿用）

## 仕様
- スロット時刻は編集可能（自動保存）
- 初期は全枠空欄
- 年切替で候補プリセットを自動読込
- 保存ボタンなし（状態は自動保存・自動復元）
- 時間軸レーン表示で、同時刻の被りが横並びで見える
- 画像出力も時間軸ベースでPNG化
